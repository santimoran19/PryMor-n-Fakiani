- Fakiani Octavio: Back End y Integración.

### 1.2 Descripción del problema

Actualmente, muchos complejos de canchas en el país, gestionan sus reservas mediante planillas de papel o chats de WhatsApp, lo que genera errores como la superposición de horarios (doble reserva) y falta de visibilidad para el cliente sobre los turnos libres. Existe la necesidad de una plataforma centralizada que permita automatizar el proceso y brindar transparencia.

### Objetivo del proyecto

Desarrollar una aplicación web integral que permita a los administradores de complejos deportivos gestionar la disponibilidad de sus canchas y a los usuarios finales realizar reservas de turnos en tiempo real, asegurando la integridad de los datos y la facilidad de uso.
